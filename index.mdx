## What is HyperID?

HyperID is a real-time facial recognition system for professional photographers at red carpet, sports, and live events. It eliminates the manual ID bottleneck — tagging talent the moment the frame is captured.

## The Pipeline

Sony Camera → LAN → Sony PDT-FP1 (Android 13, Snapdragon 8 Gen 2) → Face Crop → AWS Rekognition (97%+ confidence) → EXIF/IPTC Metadata → FTP

Recognition runs non-blocking. Capture is never interrupted.

## Primary Use Case

**Red carpet.** Shoot, ID runs in background, metadata is embedded before the image hits the FTP. No post-session tagging.

## Long-Term Target

LA 2028 Olympics — unbibbed athletes, impossible to ID manually at speed.

## Hard Requirement

Recognition must **never** interrupt capture. Non-blocking architecture is not optional.
